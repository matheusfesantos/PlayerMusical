🎵 Player Musical 🎶
Descrição:
O Player Musical é um projeto em desenvolvimento, criado por um amante da música 🎧. A proposta é oferecer uma interface inovadora e diferente do comum para a reprodução de músicas. Este projeto ainda está em fase de ideação, mas pretende se tornar uma ferramenta incrível para os usuários de dispositivos móveis 📱.

🛠 Tecnologias Utilizadas
JavaFX: Para a construção da interface gráfica.
Media Player: Para a reprodução de músicas.
🎨 Funcionalidades
Reprodução de Música: Toque suas músicas favoritas com facilidade.
Navegação: Acesse músicas anteriores e próximas com botões dedicados.
Exibição de Álbum: Veja a arte do álbum da música atual e dos álbuns anterior e seguinte.
Interface Amigável: Design focado na experiência do usuário, com texto claro e botões intuitivos.
📅 Próximos Passos
Aprimorar a interface para torná-la mais responsiva e amigável para dispositivos móveis.
Adicionar mais funcionalidades, como listas de reprodução e suporte a diferentes formatos de música.
Melhorar a animação de transição entre as músicas.
